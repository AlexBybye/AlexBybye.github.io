# 🎉 个人博客建站庆祝 - 项目功能介绍与使用指南

欢迎来到我的个人博客网站！经过一段时间的努力开发，这个基于Vue 3构建的现代化个人博客终于正式上线了。在这篇文章中，我将为大家详细介绍这个项目的功能特色和使用方法。

<img url="img/1.jpg"/>

## 🚀 项目概述

这是一个采用现代前端技术栈构建的个人博客系统，主要特点包括：

- **Vue 3 + TypeScript**: 使用最新的Vue 3框架配合TypeScript，提供更好的类型安全和开发体验
- **响应式设计**: 适配各种设备尺寸，提供良好的浏览体验
- **动态路由**: 支持多种页面导航和内容展示
- **Markdown文章**: 支持Front Matter格式的Markdown文章管理
- **动画效果**: 包含多个精美的动画页面和视觉效果

## 🛠️ 技术栈

- **前端框架**: Vue 3
- **状态管理**: Pinia
- **路由**: Vue Router 4
- **构建工具**: Vite
- **CSS预处理器**: 原生CSS配合Vue样式
- **Markdown解析**: Marked
- **代码规范**: ESLint + Prettier
- **动画库**: Canvas Confetti

## 📋 功能模块

### 1. 首页动画
- **Animation1**: 初始欢迎界面，包含背景动画

## 📝 文章管理系统

### 添加新文章
1. 在 `public/article` 目录下创建新的 `.md` 文件
2. 文件命名规则：任意，如 `曼波.md`
3. 在文件顶部添加 Front Matter 元数据：

```markdown
---
title: 文章标题
date: 2024-12-18
category: 文章分类
tags: [标签1, 标签2, 标签3]
description: 文章描述
---

# 正文内容
```

### 重新生成文章索引
运行以下命令重新生成文章列表：
```bash
npm run generate-articles
```

或者直接执行：
```bash
npx tsx scripts/generate-articles-json.ts
```

此命令会扫描 `public/article` 目录下的所有 `.md` 文件，提取元数据并生成 `articles.json` 文件供前端使用。

## 🔧 开发与部署

### 本地开发
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 构建生产版本
```bash
# 构建项目
npm run build

# 预览生产版本
npm run preview
```

### 代码质量
```bash
# 代码格式化
npm run format

# 代码检查
npm run lint

# 类型检查
npm run type-check
```
